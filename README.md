# 📌 Overview

This project focuses on detecting **hate speech**, **offensive language**, and **cyberbullying content** from text using Natural Language Processing (NLP) and Machine Learning techniques.

The model analyzes text messages and classifies them into different categories to help create safer online communities and social media environments.

---

# 🎯 Objectives

✅ Detect hate speech automatically

✅ Identify offensive language

✅ Reduce cyberbullying on online platforms

✅ Build a scalable content moderation system

✅ Apply NLP preprocessing techniques
